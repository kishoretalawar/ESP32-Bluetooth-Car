# ESP32-Bluetooth-Car
Bluetooth Controlled Car using ESP32 and L298N Motor Driver

## Project Overview
This project is a Bluetooth-controlled robotic car built using an ESP32 development board, L298N motor driver, and four DC motors.

The car can be controlled wirelessly through a smartphone using the Arduino Bluetooth Controller application.

## Features
- Forward movement
- Backward movement
- Left turn
- Right turn
- Bluetooth communication
- Wireless control

## Components Used
- ESP32 Development Board
- L298N Motor Driver
- 4 DC Motors
- Wheels
- Car Chassis
- Battery Pack
- Jumper Wires
- Switch

## Software Used
- Arduino IDE
- ESP32 Board Package
- Arduino Bluetooth Controller App

## Working Principle

- The smartphone sends commands through Bluetooth.
- ESP32 receives these commands and processes them.
- The ESP32 controls the L298N motor driver.
- The motor driver powers the four DC motors.
- This enables the car to move in different directions.
